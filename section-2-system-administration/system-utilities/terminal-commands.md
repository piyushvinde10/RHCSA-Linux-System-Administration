# ⌨️ Terminal Control Keys and Commands in Linux

Terminal control keys and commands help you **manage processes and control terminal behavior efficiently**.

---

# 📌 1. Ctrl + C (Interrupt)

Used to stop a running process.

### Example:
- Stop a running command like `ping`

✔ As shown on page 1, it stops execution immediately :contentReference[oaicite:0]{index=0}  

---

# 📌 2. Ctrl + Z (Suspend)

Used to pause a running process.

### Example:
- Suspend a command like `sleep 100`

### Output:
- Process moves to background (stopped state)

✔ Page 1 shows suspended job using Ctrl+Z :contentReference[oaicite:1]{index=1}  

---

# 📌 3. q (Quit)

Used to exit from:

- Manual pages (`man`)
- Logs (`less`, `more`)

✔ Mentioned on page 1 as quit option :contentReference[oaicite:2]{index=2}

# 📌 5. exit

Used to exit from:

- Terminal session  
- Root or user session  

### Command:

# 📌 6. script

Used to record terminal session.

### Command:
script filename.log

### Example:
script my_activity.log

### Output:
- Saves all terminal activity into file  

✔ Page 3 shows recording session into log file :contentReference[oaicite:5]{index=5}

# ⚙️ Important Notes

- Ctrl+C → Stop process  
- Ctrl+Z → Pause process  
- `q` → Exit viewers  
- `clear` → Clean screen  
- `exit` → Close terminal  
- `script` → Record session
